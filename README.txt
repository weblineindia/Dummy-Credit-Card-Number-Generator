== Description ==
Test your Payment Gateway Integration with our Test Credit Card Number generator plugin. Payment gateway developers use our program for dummy credit card number generation. Using this plugin you can have test card number generator on your website and provide your visitors to get test card numbers.

These numbers DO NOT work in real transactions! They are for testing purposes only. Use it with CVV as 123 and future expiration dates.

Our plugin uses Luhn algorithm to generate dummy card numbers. This algorithm is freely available across the web even Wikipedia.org. So go ahead and install this plugin and enable your site visitors to generate dummy card numbers from your website.

== Installation ==

1. Upload the plugin files to the \'/wp-content/plugins/plugin-name\' directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the \'Plugins\' screen in WordPress admin dashborad.

3. Use the Settings-> Credit Card Number Generator screen to configure the plugin setting.
